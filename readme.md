# Matchstick Woods
A Lightweight Wordpress Theme for Small Business

Matchstick Woods was created for Charred Wood Products, LLC and is designed to be responsive and focus on SEO improvement. This theme was not inteded to be used for blogging and is optimized for simple product image galleries with a simple contact form.
